Confirmer la demande d'exécution d'une action transmise sous la forme d'une expression lambda 

confirmer(() => { write((msg', genererMessage('Expression lambda exécutée', 'vert'))} );

Confirmer la demande d'exécution d'une action transmise sous la forme d'un pointeur de fonction 

confirmer(maFonction);
function maFonction() {
    write('msg2', genererMessage('Fonction maFonction exécutée', 'vert'));
}
