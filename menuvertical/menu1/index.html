Vous êtes dans le menu vertical, dans le dossier menu1.

Le fichier personnalisé /menuvertical/include/interface.php charge le menu vertical.
$lesOptionsVerticales = file_get_contents(RACINE . '/menuvertical/.config/menuvertical.json');
Il active le composant depuis une balise script
import {initialiserMenuVertical} from "/composant/menuvertical/menu.js";
initialiserMenuVertical($lesOptionsVerticales, 230);
Le paramètre 230 correspond à la largeur du menu.
Le fichier de configuration menuvertical.json est lu par le script PHP et injecté dans la variable $lesOptionsVerticales.
